LWSWS server Procursus default page - It works!

It works!

The LWSWS web server has be correctly set up to accept requests from localhost

What to do next

In @MEMO_PREFIX@/etc/lwsws/conf.d/example.conf, replace localhost with your actual domain name such that the server could correctly serve requests from there.

Files placed at @MEMO_PREFIX@/var/www/lwsws-default will be served over plain text HTTP to localhost.
